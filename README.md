# Mini-CSS

**_Mini-CSS_** is a simple compiler that compile css into minified css and combine css property.

---

### For example

> input code

```css
body {
  color: blue;
}
span {
  color: blue;
}
```

<br />

> output code

```css
body,
span {
  color: blue;
}
```

---

## How To Use:

> requirements

- nodejs
- npm

```
$ git clone https://github.com/MohamedElmdary/mini-css
```

```
// npm
$ npm install

// yarn
$ yarn install
```

```
// npm
$ npm run test
$ npm run example

// yarn
$ yarn test
$ yarn example
```
